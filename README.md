📌 XBOX GAME PASS SUBSCRIPTIONS SALES – Dashboard de Vendas

🎯 Objetivo do Projeto
Este projeto foi desenvolvido como parte de um desafio de análise de dados, com o objetivo de transformar uma base bruta de assinaturas do Xbox Game Pass em um dashboard visual, organizado e interativo no Excel. O painel permite analisar receita, comportamento dos assinantes, tipos de planos, passes adicionais e evolução temporal das vendas.

📊 Descrição dos Dados <br>
A base contém informações de assinantes, incluindo:
- Subscriber ID
- Name
- Plan (Core, Standard, Ultimate)
- Start Date
- Auto Renewal (Yes/No)
- Subscription Price
- Subscription Type (Monthly, Quarterly, Annual)
- EA Play Season Pass (Yes/No)
- Minecraft Season Pass (Yes/No)
- Coupon Value
- Total Value (receita final por assinatura)

Esses dados permitem análises financeiras, comportamentais e comparativas.  
Além da base, foi feita uma guia para organizar os elementos visuais para a construção do dashboard, como:
- Cores
- Imagens
- Logos
- ícones

🧮 Metodologia  
O dashboard foi construído utilizando:
- Tabelas Dinâmicas
- Gráficos Dinâmicos
- Colunas auxiliares (ex.: Trimestre)
- Segmentações (quando aplicável)
- Paleta de cores personalizada inspirada no Xbox

Foi criado uma guia com as tabelas dinâmicas para gerar os cálculos necessários para os dashboards.  
Alguns dos cálculos incluídos:
- Receita total
- Receita por tipo de plano
- Receita por tipo de assinatura
- Receita por auto-renovação
- Receita por mês, trimestre e semestre
- Receita com e sem passes adicionais
- Contagem total de assinaturas

Após a criação dos cálculos, foi gerado os gráficos para analisar os dados.

📈 Análises Realizadas no Dashboard  
KPIs Principais:  
- Receita total
- Número total de assinaturas
- Receita do EA Play Season Pass 
- Receita do Minecraft Season Pass

Gráficos e Visualizações
- Revenue With and Without Season Pass
- Auto-Renewal Revenue Comparison
- Monthly and Quarterly Revenue Trend
- Revenue by Subscription Type
- Revenue by Plan (Core, Standard, Ultimate)
- Season Pass Impact on Revenue

🔁 Como Reproduzir  
Baixe o arquivo XBOX GAME PASS SUBSCRIPTIONS SALES.xlsx  
Abra no Excel  
Habilite edição, se solicitado  
Atualize as tabelas dinâmicas caso necessário  
Navegue pelas abas e interaja com os gráficos

O dashboard permite visualizar claramente:  
O impacto dos passes adicionais na receita
